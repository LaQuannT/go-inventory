Inventory Management System

command flags:
  - a: add item
  - d: delete item
  - s: search item by sku(stock keeping unit)
  - n: search item by name
  - b: search all data from a brand
  - e: edit item data
  - c: search item by name and give count

item table columns:
  - name
  - brand
  - sku
  - location
  - product type
